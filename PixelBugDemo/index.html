Play Mode

Play Mode is a separate scene mockup workspace. Use actor frame selectors to choose idle, walk, and jump poses. Use scene size fields to set the stage. Use place frame art controls to add objects from any frame. Objects can be dragged on the scene canvas, scaled, deleted, marked solid for collision, or marked ghost to ignore collision. Run starts a simple sidescroller preview using keyboard movement.

Scene canvas. Click empty space to set the next object position. Click an object to select it. Drag a selected object to move it. Use A, D, left arrow, or right arrow to move the actor while running. Use Space, W, or up arrow to jump.

Start or stop the playable scene preview. Start the selected nearby character dialogue in Play Mode. Keyboard shortcut E. Export the current Play Mode scene as a PNG image. Export a short animated Play Mode scene preview as a GIF. Export a standalone HTML game that can be opened in a browser and shared with friends.

Brightness modes use color value to vary extrusion depth.

Uniform depth gives all pixels the same extrusion. Brightness height map raises brighter colors and recesses darker colors. Reverse brightness height map raises darker colors and recesses brighter colors. Landscape terrain treats blues as low water, greens as rolling land, browns and grays as higher ground, and light colors as peaks. Layer depth steps gives each visible layer its own depth tier, so it is most visible when multiple layers overlap.
0° Rotates the voxel model around the vertical depth axis, useful for landscape previews and angled map views.

Privacy Policy

Pixel Bug stores your work locally.

Autosave uses your computer's browser-style localStorage inside Electron. Projects, frames, layers, colors, and palette presets stay on this device unless you export or share files yourself.

Image pixelizing happens locally in the app. Uploaded images are read into the editor for preview/import and are not sent to an online service by this app.

Saved project files, PNGs, spritesheets, and GIFs are only created when you choose a save or export option.

Uninstalling the app or clearing app storage may remove the local autosave. Use Save to keep a separate project file.
